# header

The `header` predicate checks the value of a predicate.

## Options

| Option  | Required | Description        |
| ---     | ---      | ---                |
| `name`  | yes      | Name of the header |
| `value` |          | Value to match     |

## Results

| Field    | Type    | Description            |
| ---      | ---     | ---                    |
| `result` | boolean | true is header matches |
